webpack -- webpack-dev-server启动时报错

发表于 2018-07-19
字数统计: 415字 | 阅读时长 2分钟

显示如下报错

解决方案

  1. 我的webpack是3.6版本的，
  2. 我查了一下我webpack-dev-server版本是3.1.4的版本，
  3. 这两个版本不兼容，把webpack-dev-server降到2.x版本就好使了
