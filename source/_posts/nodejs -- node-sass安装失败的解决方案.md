node-sass安装失败的原因是被墙了；

解决办法是翻墙或者使用淘宝镜像cnpm下载；

```
cnpm install node-sass -D
```

cnpm的安装和使用方法链接：[cnpm淘宝镜像的安装和使用方法](https://blog.csdn.net/wxl1555/article/details/71172285)
